Дискриминант:

     • Дискриминант – это математическое выражение, определяющее количество корней квадратного уравнения вида ax² + bx + c = 0
     
         • Формулы:
         
              D = b² - 4ac
              x = -b ± √D / 2a (если D > 0)
              x = -b / 2a (если D = 0)
         
         • Свойства:
         
              Если D равен меньше нулю, то он не имеет решений
              Если какая-то буква отсутствует (например ax² - bx) то она будет равна 0 (в этом случае c будет равно 0)
